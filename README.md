# JavaDumper

### dump of all classes of the active java process

## How use?
### 1. Download Dll
### 2. Inject to the desired java process
### 3. Open cmd and enter "curl.exe --http0.9 -X POST http://localhost:2716 -d "/dci --header ?ux ?iof of=FILE_NAME.txt""
### After that, a file with your name will be created in the root folder of your process with all the classes.
